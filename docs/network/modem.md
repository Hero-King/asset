# 光猫


## 桥接

1. 取消 hdcp 使能或者关闭 dhcp 服务器
2. 记录好 Internet 连接和 iptv 连接 vlanid
3. internet业务改成桥接

## vlan 绑定

选择好连接和绑定的某个端口, 输入用户侧 vlanid; 然后在路由器上使用对应的 vlanid 接受数据 
`vlan绑定端口1后系统会自动取消网络连接中的业务到端口1的绑定`

## 光猫交换机功能(省掉弱电箱中的交换机)

将光猫不需要的几个端口绑定到一起, 主路由有线回程到某一个不需要的端口
